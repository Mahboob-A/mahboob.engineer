## Strong Fit

Mahboob is strongest for backend and platform roles where product ownership
meets infrastructure: APIs, async workers, queues, caching, databases,
deployment automation, and production debugging.

## Good Interview Topics

- Designing a Django or FastAPI backend from scratch.
- Redis caching and queue-backed workflows.
- PostgreSQL schema and query optimization.
- Running untrusted code safely.
- Video processing pipelines.
- CI/CD and cloud cost reduction.
- Tradeoffs between SSE, WebSocket, Celery, RabbitMQ, and REST.
